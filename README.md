Dynamic Portfolio Website using Django

Developed a fully dynamic Developer Portfolio Website using Django framework.

The project includes an admin dashboard to manage projects, images, tech stack, and project links. Built with responsive Bootstrap UI and integrated SQLite database for backend storage.

Key Features:
• Admin-controlled project management
• Dynamic project showcase section
• Image upload functionality
• GitHub & Live project links
• Responsive design
• MVC architecture implementation

Tech Stack: Python, Django, HTML, CSS, Bootstrap, SQLite
